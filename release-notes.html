🎉 Granulay 1.1.2 Release!

Nova versão estável do Granulay com interface simplificada e sistema de atualizações aprimorado!

✨ Novidades na versão 1.1.2:

  • 🔧 Interface simplificada - Removido o toggle de salvamento automático para uma experiência mais limpa
  • ⚡ Configurações otimizadas - Sistema de persistência de configurações mais eficiente e confiável
  • 🔄 Sistema de atualizações melhorado - Correção do canal de atualizações para detectar versões estáveis
  • 🔔 Notificações inteligentes - Lembretes suaves para atualizações em segundo plano com badges no Dock

✨ Recursos existentes:

  • 🎨 Efeito granulado vintage - Adiciona textura nostálgica em toda a tela
  • 🎛️ Intensidade ajustável - Controle preciso do nível de granulação
  • 🌈 Múltiplos estilos de grão - Diferentes texturas para personalizar sua experiência
  • ⚡ Performance otimizada - Renderização eficiente sem impacto na performance
  • 🎯 Interface na barra de menu - Acesso rápido e discreto às configurações
  • 🔄 Atualizações automáticas - Sistema inteligente de atualizações em segundo plano
  • 🌍 Suporte multilíngue - Interface em português e inglês

🔧 System Requirements:

  • 💻 macOS 13.0 ou superior
  • 🖥️ Arquitetura: Intel x64 e Apple Silicon (M1/M2/M3)
  • 💾 Espaço em disco: ~6 MB

Versão Estável: Esta é uma versão estável e pronta para uso diário. Aproveite a experiência vintage aprimorada!
