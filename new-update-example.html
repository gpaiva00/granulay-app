🎉 Granulay 1.0.4-beta Beta Release!

Nova versão beta do Granulay com melhorias no sistema de atualizações!

✨ Novidades na versão 1.0.4-beta:

  • 🔧 Sistema de atualizações melhorado com suporte ao canal beta
  • 📝 Logs mais detalhados para debugging
  • 🛠️ Correções gerais de estabilidade

✨ Recursos existentes:

  • 🎨 Vintage grain effect for the entire screen
  • ⚙️ Customizable settings for intensity and style
  • 🖥️ Multi-monitor support with individual settings
  • 🔧 Menu bar integration for quick access
  • 💡 Brightness preservation option
  • 🎯 4 grain styles: Fine, Medium, Coarse, Vintage

🔧 System Requirements:

  • macOS 13.0 (Ventura) or later
  • Apple Silicon or Intel Mac
  • 4GB RAM minimum

Beta Notice: This is a beta release. Please report any issues to our support team.
